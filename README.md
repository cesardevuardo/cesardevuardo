# ¡Hols!

Esta cuenta está dedicada a centralizar distintos proyectos, principalmente como soporte para una serie de tutoriales relacionados con conceptos básicos para el desarrollo web y el uso de sus herramientas, aunados en una SAP React/NodeJS, apuntando a resolver (en parte...) el uso 'inicial' de esas herramientas, Git, GitHub, Visual Studio Code, GitHub-Pages, Netlify, Vercel, Firebase, Postman y algunas otras son las temáticas a recorrer, respecto de las tecnologías en sí, se veran en principio: HTML, CSS, JavaScript y React, aunque no es la intención de éste proyecto profundizar en la explicación de lenguajes de código sino facilitar un mejor entendimiento del manejo usual de algunas de las herramientas "freeware" necesarias en distintos casos.

No pretendo que se trate de una exposición exhaustiva de los temas, ya que ello estaría muy lejos de los alcances de esta iniciativa, de mis posibilidades y de mis conocimientos, la intención del laboratorio es mas bien un acercamiento amigable a los conceptos iniciales necesarios para el uso mas elemental de estas herramientas y tecnologías, en español rioplatense..
