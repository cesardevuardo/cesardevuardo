Esta cuenta está dedicada a centralizar distintos proyectos entre los que se encuentra una serie de tutoriales relacionados con conceptos básicos para el desarrollo web apuntado al uso de herramientas necesarias para el mismo, incluyendo: Git, GitHub, Visual Studio Code, GitHub-Pages, Netlify, Vercel, Firebase y algunas otras, respecto de las tecnologías en sí, se veran en principio: HTML, CSS, JavaScript y React, aunque no es el fin de éste proyecto desplegar cursos de código sino facilitar un mejor entendimiento del manejo usual de las herramientas necesasrias en los distintos casos de uso
