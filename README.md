Esta cuenta está dedicada a centralizar distintos proyectos, principalmente sostiene una serie de tutoriales relacionados con conceptos básicos para el desarrollo web apuntando a resolver el uso inicial de las herramientas necesarias para el mismo, incluyendo: Git, GitHub, Visual Studio Code, GitHub-Pages, Netlify, Vercel, Firebase y algunas otras, respecto de las tecnologías en sí, se veran en principio: HTML, CSS, JavaScript y React, aunque no es el fin de éste proyecto desplegar cursos de código sino facilitar un mejor entendimiento del manejo usual de las herramientas necesarias en distintos casos
